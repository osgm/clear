# Contributing to JoyCleaner

感谢你愿意为 `JoyCleaner` 做贡献。

## 贡献方式

- 提交 Bug 报告（Issue）
- 提交功能建议（Issue）
- 提交代码修复或新功能（Pull Request）
- 改进文档、测试、构建流程

## 开发环境

- Node.js `>=18`
- npm `>=9`

安装依赖：

```bash
npm install
```

开发调试：

```bash
npm run dev
```

类型检查：

```bash
npm run typecheck
```

Windows 打包：

```bash
npm run build:win
```

## 提交规范

- 保持改动聚焦：一个 PR 只做一类事情
- 代码风格与现有项目一致
- 新增功能尽量附带基本测试或验证步骤
- 不要提交本地环境文件、密钥或凭据

## Pull Request 要求

请在 PR 描述中说明：

- 变更目的（Why）
- 主要改动（What）
- 验证方式（How to test）
- 潜在风险与回滚建议（Risk / Rollback）

## Issue 报告建议

请尽量提供：

- 操作系统版本
- 复现步骤
- 期望行为与实际行为
- 截图/日志（如有）

## 法律与授权

通过提交代码，你同意你的贡献以项目当前许可证（Apache-2.0）发布。
